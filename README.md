The Tic-Tac-Toe Game is a classic two-player board game implemented as a web-based application. In this version, the player competes against a simple AI opponent. The game follows standard Tic-Tac-Toe rules and provides real-time status updates for turns, wins, and draws.

This project is developed using HTML, CSS, and JavaScript and is ideal for beginners to understand game logic, DOM manipulation, and basic AI behavior.
